### :gear: Como executar

```bash

    # Clonar o repositório
    $ git clone https://github.com/arielff3/ze-code-challenge

    # Navegar para o diretório
    $ cd ze-code-challenge

    # Baixar dependências
    $ yarn install

    # Executar Projeto
    $ yarn start

    # Realizar Testes
    $ yarn test

    # Gerar Coverage
    $ yarn test:coverage

```
